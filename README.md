# Sunshine (Clima)
Simple weather forecast app to get current weather from local or another city and country. API responds with a list of results that match a searching word.

## Built With

*  Swift 4.0
*  Xcode 9.0
*  Open weather API - https://openweathermap.org/api
*  Alamofire 4.7
*  SwiftyJSON 4.1
*  SVProgressHUD 2.2
